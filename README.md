# 🌍 WanderVista — Full-Stack Vacation Booking Platform

WanderVista is a robust vacation booking platform engineered for a seamless user experience. It allows entrepreneurs and travelers to list, discover, and reserve stays with advanced search capabilities and secure workflows.

### 🔗 [Live Demo: WanderVista](https://wandervista-dt18.onrender.com/)

---

## 🚀 Key Features
- **Secure Authentication:** Multi-layered auth using JWT and GitHub OAuth for a friction-less login experience.
- **Advanced Fuzzy Search:** Integrated `Fuse.js` to enable high-performance property searches, handling typos and partial matches effortlessly.
- **Interactive Geolocation:** Map-based location visualization using `MapTiler` API for better property discovery.
- **Robust CRUD Operations:** Full management system for property listings, user reviews, and ratings.
- **Scalable Backend:** Built with a focus on structured logic and clean architectural patterns.

## 🛠 Tech Stack
- **Frontend:** EJS (Embedded JavaScript), JavaScript (ES6+), Bootstrap, CSS3, HTML5
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose (ODM)
- **APIs & Tools:** Fuse.js, MapTiler, Passport.js, Cloudinary (for image management)

## 📂 Project Structure
- `models/`: Database schemas for Users, Properties, and Reviews.
- `routes/`: Express routers for clean endpoint management.
- `controllers/`: Logic-heavy functions separating concerns from routes.
- `views/`: Dynamic EJS templates.
